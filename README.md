# 6102TA_02 — Colecciones y comprensión

Resolución de la actividad **La Profecía del Gran Diccionario Perdido**, centrada en el uso de colecciones y comprensiones en Python.

## Autor

Jaime Martret

## Contenido

El cuaderno `6102TA_02_Martret_Jaime.ipynb` contiene la resolución razonada de:

- Misión 1 — El Caldero de las Pociones
- Misión 2 — El Registro de Dragones
- Misión 3 — Los Fantasmas Contables
- Misión 4 — El Bosque de los Unicornios
- Misión 5 — La Cripta Prohibida
- Misión 6 — La Escoba Rebelde
- Misión final — El Demonio de la Comprensión

En cada misión se analiza el código original, se identifica el error, se propone una corrección y se comprueba el resultado.

## Requisitos

- Python 3
- Jupyter Notebook o Visual Studio Code con la extensión Jupyter

## Ejecución

1. Abrir `6102TA_02_Martret_Jaime.ipynb`.
2. Seleccionar el kernel `Python 3.13 - Colecciones 6102`.
3. Ejecutar las celdas en orden.
